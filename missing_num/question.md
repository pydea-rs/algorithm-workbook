Write missing_number(nums) — nums contains n distinct numbers from the range 0..n (so exactly one of the n+1 values is missing); return the missing one. Solve it in O(n) time and O(1) space. There are two one-liner families — arithmetic (expected sum minus actual sum) and XOR — and the interviewer will enjoy hearing you name both. This drill is also your bridge into Module 2b's bit tricks.
